Meu primeiro formulário

Todo formulário deve estar dentro da tag "form" de HTML.

Para fins de localização para o navegador. Devemos sempre identificar os inputs.
Para o caso do PHP e HTML e mais comumente utilizado o "name", e para JavaScript ou Python, "id".

Não se cria inputs sem relacionar eles a um label, ou se não irá dificultar para os navegadores de entender o seu conteúdo.
